Kip.org is an effort to bring an online tool that is useful for the creative thinkers and problem solvers.
Welcome to Kip's project portal.

Kip projects run off volunteerism and charitable giving, so please take part in our mission.
Contributions can range from content editing, project ideas, project coding, engineering ideas, or sciencitific lab ideas. If you want to browse our current projects to see if you can find a place where you fit in look through our current projects below.

If you are a representative of a company please contact us using our form cabinet and follow the instructions for businesses.

Current Projects:
  1. Web Standards Project
    The Web Standards Project maintains a code base to help developers learn and build there web applications. We also support and develop concepts to make the web a better tool for scientists and engineers.
  2. Kip Science Project
    The Kip Science Project maintains resources for you to get involoved in Kip Science Labs or to create your own science labs.
  3. Kip Engineering Project
    The Kip Engineering Project maintains resources for you to get involoved in Kip Engineering or to help create your own engineering projects.
  4. University of Nebraska Outreach Project
    The University of Nebraska Outreach Project is a set of resources for student and faculty ran science and engineering organizations within the University of Nebraska education system. It is part of our community outreach project.
  5. Mozilla Project
    Kip is proud to be a part of Mozilla and its mission to teach web skills to people around the world. Please visit Kip's Mozilla project page to get more information on how we help Mozilla accomplish their mission.
